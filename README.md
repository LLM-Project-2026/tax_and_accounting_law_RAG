# RAG_Legal

RAG асистент за българското счетоводно и данъчно законодателство.

Курсовият проект е разделен на стейджове, всеки от които е напълно
самостоятелен — собствени зависимости, собствени модули, собствен
изход — и води към общия пайплайн.

## Структура

```
RAG_Legal/
└── stage_1/          ← Сваляне + трансформация (stage 1)
    ├── ingest/        нормативна уредба → chunks.jsonl
    ├── reports/       Отчети върху данните
    └── data/          суров HTML + парснати чънкове
```

## Бърз старт

```bash
cd stage_1
py -3 -m pip install -r requirements.txt
py -3 run_pipeline.py
```

Подробни инструкции — `stage_1/README.md`.
